* [Steps](https://github.com/Xilinx/kria-dfx-apps/blob/xlnx_rel_v2022.1/README.md) to build the applications. Applications for DFX example design should be built on the target. 
* [Steps](./run_application_on_target.md#install-pre-built-firmware-on-target) to install pre-built firmware on the target.
* [Steps](./run_application_on_target.md#load-pre-built-accelerators-on-the-target) to load pre-built accelerators on the target.
* [Steps](./run_application_on_target.md#run-applications-for-pre-built-accelerators) to run applications for pre-built accelerators on the target.
* [Steps](./creation_of_new_RM.md) to build a new user RM.
* [Steps](./creation_of_new_RM_application.md) to build a new user RM application
* [Steps](./run_application_on_target.md#run-new-accelerator-rm-application-on-target) to load and run a new user RM application
